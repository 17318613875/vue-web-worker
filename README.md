# vue-web-workers

A Vue component to use Web Workers in a simply way

## Install

```shell
yarn add vue-web-workers
npm install vue-web-workers
```

### Use

```shell
yarn serve
```

### API

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
